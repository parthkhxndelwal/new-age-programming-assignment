# NEW AGE PROGRAMMING – Assignment Snapshot

## Overview of Go, F#, Clojure, Kotlin
- **Go:** Concurrency via goroutines, simple syntax, standard tooling for cloud-grade services.
- **F#:** Functional-first language on .NET with type inference, pattern matching, and seamless C# interop.
- **Clojure:** Dynamic Lisp on the JVM using persistent data structures, macros, and Java ecosystem access.
- **Kotlin:** Modern JVM language with null safety, coroutines, concise syntax, and full Java compatibility.

## Comparison with Traditional Languages
Go, F#, Clojure, and Kotlin emphasize safety, succinctness, and managed runtimes versus C/C++/Java’s manual memory or verbosity. Detailed feature alignment, including typing, paradigms, concurrency, and interop, is captured in `docs/comparison-table.md`.
